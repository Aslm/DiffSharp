The values are normalized with respect to the running time of the original function corresponding to each column. Operations diffn and diffn' are used with \(n=2\), for enabling comparisons with diff2 and diff2'.

The used functions were \(f(x) = (\sin \sqrt{x + 2}) ^ 3\) for the scalar-to-scalar case, \(f(x,y,z) = (x\;\sqrt{y + z}\;\log z) ^ y\) for the vector-to-scalar case, and \(f(x,y,z) = (\sin{x ^ y}, \sqrt{y + 2}, \log{xz})\) for the vector-to-vector case.

The running times were measured using Process.TotalProcessorTime, averaged over a million calls to each operation, on a PC with an Intel Core i7-4510U 2.0 GHz CPU and 16 GB RAM, running Windows 8.1 and .NET Framework 4.5.1.

Running Benchmarks on Your Machine

If you would like to run the benchmarks on your own machine, you can use the command line DiffSharp Benchmarks tool distributed together with the latest release on GitHub.
